
# Playwright

## Electron-based 
### nvm & nodejs
#### macOS
```shell

# install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

nvm install --lts # using long-term support
nvm use --lts # using long-term support
```
```shell
# check
nvm -v
node -v
```
